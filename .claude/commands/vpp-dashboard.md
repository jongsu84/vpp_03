# 📋 AI-VPP 2.0 대시보드 관리 가이드 (vpp-dashboard.md)

이 문서는 클로드 에이전트가 `index.html` 파일을 수정하거나 데이터를 업데이트할 때 준수해야 할 표준 규칙입니다.

---

## 1. 프로젝트 개요
- **플랫폼 명칭:** AI-VPP 2.0 (통합 관제 플랫폼)
- **파일 구성:** `index.html` (단일 파일 기반 대시보드)
- **주요 목적:** 실시간 전력 데이터 시각화 및 계통 상태 모니터링

## 2. 데이터 업데이트 규칙
에이전트는 사용자 요청 시 아래의 규칙에 따라 `index.html` 내의 SVG 및 텍스트 데이터를 수정한다.

### 2.1 텍스트 수치 (Text Elements)
- **실시간 총 출력:** `148.7 MW`와 같이 표시된 수치를 사용자가 지정한 값으로 업데이트한다.
- **주요 지표:** 계통 주파수(60Hz), SMP 수치 등 텍스트 노드의 값을 정확히 매칭하여 수정한다.

### 2.2 실시간 그래프 (SVG Polyline)
- **그래프 수정:** `<polyline>` 태그의 `points` 속성값을 수정하여 데이터 추세를 시각화한다.
- **좌표 계산:** - x축: 시간 흐름에 따른 가로 위치
  - y축: 출력량 (값이 클수록 아래로 내려가는 SVG 특성을 고려하여 계산)

## 3. 디자인 가이드라인 (CI/BI)
에이전트는 수정 시 기존의 디자인 일관성을 유지해야 한다.
- **핵심 색상:** `#0059ff` (기본 파란색), `#1f98ff` (보조 파란색)
- **강조 색상:** `#ffca42` (상태 알림 노란색)
- **레이아웃:** 카드 형태의 섹션 구조와 그림자 효과를 변경하지 않는다.

## 4. 에이전트 실행 지침 (Action Items)
1. **분석:** 수정 요청이 들어오면 `index.html`에서 해당 데이터가 위치한 태그를 먼저 찾는다.
2. **수정:** MD 가이드의 규칙을 준수하여 코드를 변경한다.
3. **확인:** 수정된 HTML 코드가 문법적으로 올바른지 확인하고, 변경된 부분을 사용자에게 요약하여 보고한다.
